# Pagekin — Development Roadmap

> AI-powered personalized children's storybook with interactive mode, voice recording, and PDF export.

---

## App Concept
Pagekin lets parents and children create unique, personalized AI-generated stories. Enter a child's name and a theme, choose linear or interactive mode, and Pagekin generates a beautifully illustrated story page by page. Parents can record their own voice for each page, and the finished story can be downloaded as a PDF keepsake.

---

## Screens & Flow

```
Home Screen
  → Enter child's name
  → Enter story theme / description
  → Choose page count (5 / 10 / 15 / 20)
  → Toggle: include child in story
  → Choose mode: Linear or Interactive
  → Click "Generate Story"

Loading Screen
  → "Writing your story..." animation
  → Generates page 1 text via Gemini
  → Remaining pages generated on demand

Story Screen (Linear Mode)
  → AI illustration per page (HuggingFace FLUX.1)
  → Story text (child's name woven in if toggled)
  → Read Aloud button (Gemini TTS) with loading indicator
  → Record button (per page, sequential)
  → Prev / Next navigation
  → "Save as PDF" button
  → "New Story" button back to Home

Story Screen (Interactive Mode)
  → Same as linear but after each page:
  → Show 2-3 AI-generated choices OR free text input
  → Next page generated based on user's choice
  → Story ends after chosen page count
  → "Save as PDF" button at the end

PDF Export
  → Includes all page text + AI illustrations
  → Child's name on cover
  → Story title generated by AI
```

---

## Tech Stack
- **React + TypeScript** (Vite)
- **Gemini 2.5 Flash** — story text generation, chat, TTS
- **HuggingFace FLUX.1-schnell** — illustration generation
- **Vercel** — hosting + serverless API proxy
- **jsPDF** — PDF export

## Known Issues / Notes
- HuggingFace free tier has limited monthly credits (~$0.10 included)
- Imagen 4 requires Google Cloud billing — not viable for free tier
- Gemini TTS + chat work on free tier via proxy
- Images cached per page to avoid redundant API calls
- ApiKeyBanner triggers on 429 errors for graceful degradation
